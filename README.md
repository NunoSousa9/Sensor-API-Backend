# Sensor-API-Backend

## Overview

This is the backend component of the **Sensor API Management Application**. It handles RESTful requests for sensor data, processes asynchronous messages with RabbitMQ, and manages database interactions with MongoDB.

## Main Repository

This backend service is part of the main [Sensor_API](https://github.com/NunoSousa9/Sensor_API) project.

## Technologies Used

- Java
- Spring Boot
- Spring Data MongoDB
- Spring Security and OAuth2
- RabbitMQ
- Feign
- Swagger
